CÓDIGO PARA LEER JSON QUE EMPIEZA POR {}

en un datahandler.java... (Clase es una clase externa, con lo de constructor (vacío y con parámetros), getter y tostring... En el pom tiene que estar json) 
Luego en el ejecutable hacemos Datahandler.readClase()


CÓDIGO:
public static void readClase(){

	Gson gson = new Gson();

	try {
	
		Clase clase = gson.fromJson(new FileReader("archivojson.json"), Clase.class);
		System.out.println(clase);

	} catch (FileNotFoundException e) {
		throw new RuntimeException(e);
	}
}
